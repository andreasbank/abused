18 ## Get up and running
19 
20 Get up and running in no time:
21 
22 - Clone the repo: `git clone https://github.com/andreasbank/libssdp.git`
23 - Enter the directory `cd libssdp`
24 - Build `make`
25 - Run `./ssdpscan -U` and get results

125 ## Documentation
126 
127 libSSDP's documentation is embedded in all the source and header files as doxygen comments and also available as html here [documentation](http://andreasbank.github.io/libssdp).
128 
129 ### Build the documentation locally
130 
131 You can even build the documentation yourself. The prerequisites are `doxygen` and `dot`. Once this is in place simply run `make docs`. This is recommended since the online documentation might be outdated or in the process of being rebuilt.
